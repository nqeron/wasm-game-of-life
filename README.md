## About

This project uses Rust to compile WebAssembly program which implements Conway's Game of Life.


### 🛠️ Build with `wasm-pack build`

```
wasm-pack build
```

### 🎁 Publish to NPM with `wasm-pack publish`

```
wasm-pack publish
```
